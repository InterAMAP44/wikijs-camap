votre compte producteur

Actuellement, en tant que membre des groupes amap, vous avez la possibilité de bénéficier de certains droits sur votre contrat ou pas.

Avec ce nouveau développement de camap vous aurez accès à un compte producteur unique, partagé entre les différentes AMAP où vous effectuez des distributions. Ce compte vous offrira une connexion centralisée à CAMAP, avec votre adresse e-mail comme identifiant.

Dans notre vidéo explicative de 8 minutes, nous vous présentons étape par étape  la revendication de votre compte et vous guiderons pour une utilisation optimale de votre nouvel espace producteur sur CAMAP.

 

Revendiquer

Revendiquer signifie affirmer qu’une chose vous appartient ou vous revient de droit.

Dans ce contexte, revendiquer votre exploitation agricole consiste à déclarer officiellement que cette exploitation est bien la vôtre et que vous en êtes le propriétaire ou l’exploitant légitime.

Il est possible que, dans la base de données de la CAMAP, vous apparaissiez sous deux ou trois identités différentes. Dans ce cas, ces différentes identités vous seront proposées. Elles seront fusionnées suite à votre revendication, et vous serez alors le seul habilité à la modifier.

Après avoir revendiquer, Les comptes de chaque AMAP seront accessibles à partir du compte producteur.

Votre compte CAMAP

Si vous n'avez pas encore accès à CAMAP ou n'en êtes pas certain, veuillez suivre ces étapes pour créer ou récupérer vos identifiants.
Pour accéder à CAMAP,  vous recherchez le terme CAMAP dans votre moteur de recherche. Une fois sur la page d'accueil, cliquez sur "Se connecter" ou  "S'inscrire" 
Remplissez le formulaire d'inscription en le complétant .Si vous êtes déjà enregistré, ce qui est probable, vous pouvez cliquer sur "Mot de passe oublié" pour générer un nouveau mot de passe grâce au lien envoyé par e-mail.
Nous vous recommandons de vérifier attentivement votre dossier de courriers indésirables ou de spam si vous ne recevez pas le message dans votre boîte de réception principale.
Ces étapes simples vous permettront d'accéder à votre compte.
