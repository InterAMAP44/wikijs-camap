Gestion des absences

Pour chaque contrat, le coordinateur avec l'accord du producteur peut définir ou pas une plage de temps pendant laquelle les amapiens pourront être absent, ainsi qu'un nombre maximum d'absences autorisées. Les amapiens n'auront donc pas de commande enregistrée pour ces jours d'absence, c'est une sorte de “joker” .

 

* Contrats AMAP classiques : la gestion des absences est présente par défaut sur la gestion du contrat, onglet à gauche

* Contrats AMAP variables : la gestion des absences n'est disponible que lorsque l'option “Minimum de commande par distribution” est activée dans les paramètres du catalogue ( minimum supérieur à zéro ). Si cette option n'est pas activée, cela veut dire que la commande n'est pas obligatoire à chaque distribution, il n'y a donc pas lieu de gérer des absences.

Lorsque l'amapien saisira sa souscription au contrat, il pourra choisir le nombre de distributions pour lesquelles ils sera absent ( entre zéro et le maximum autorisé ). Attention, une fois la souscription créée, il ne pourra plus modifier le nombre de jours d'absences et devra demander au coordinateur de contrat de faire une modification.  Dans l'autre exemple ci dessous le producteur de fraises remontantes autorise 3 absences entre le 11 Mai et le 30 Septembre et Aurelie aura retenue que 2 absences sur les 3 absences possibles, seul le coordinateur pourra rajouter le nombre d'absences possible entre 0 et 3 ici.

 

Par contre les dates d'absences pour chaque amapien sont modifiables dans son espace onglet  “Mes contrats”.  La date d'absence pour une distribution n'est plus modifiable quand la date de fermeture des commandes est dépassée. 

Chemin d'accès pour Aurelie dans son espace:   MES CONTRATS  /  MODIFIER MA SOUSCRIPTION  / MODIFIER mes dates d'absences  

 

Informations pour les coordinateurs pour gérer les absences:

Voici les 3 difficultés que l'on peut rencontrer:

1 - Le souscripteur n'a pas vu l'écran pour choisir des dates d'absence en laissant le "0" par défaut:  dans ce cas le souscripteur retourne sur sa souscription en cliquant sur le catalogue de la page d'accueil, en bas en rouge il y a le bouton "Annuler la souscription"  ou bien c'est le coordinateur avec les droits admin: SOUSCRIPTION / GÉRER / SUPPRIMER

2 - Le souscripteur  veut modifier ses dates d'absences, il suffit d'aller dans  MES CONTRATS / MODIFIER MA SOUSCRIPTION / MODIFIER mes absences

3 - Le souscripteur s'est trompé de nombre de jours d'absence, dans ce cas seul le coordinateur peut faire la modification : SOUSCRIPTION / GÉRER / MODIFIER
