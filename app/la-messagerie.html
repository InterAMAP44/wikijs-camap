La Messagerie

Y accéder

Le menu Messagerie est accessible aux utilisateurs ayant les droits d'administration  “Messagerie”  (à configurer dans le menu Paramètres du groupe).

Expéditeur

Le courrier de l'expéditeur est renseigné automatiquement lorsque vous accédez au menu Messagerie, mais vous pouvez le changer. 

Destinataires

Vous pouvez sélectionner les destinataires du message parmi des listes de diffusion préétablies (voir la section suivante dans le sommaire)
ou bien vous pouvez vous-même ajouter les adresses e-mail de vos destinataires.

Limitations

Le poids total du message ne peut pas dépasser 15Mo. Ceci inclus le poids des images et le poids des fichiers joints.

La pièce jointe est encodée pour des raisons techniques, et le poids résultant dans l'e-mail est environ 1,3x supérieur au poids du fichier initial.

Certains formatages peuvent mal fonctionner si vous copiez/collez votre message depuis un logiciel tierce (LibreOffice, Microsoft Word, Google Doc, etc). Si cela arrive, vous pouvez toujours formater les éléments que vous voulez “à la main” dans la messagerie CAMAP. Voici une liste non exhaustive de ces limitations :

Les couleurs ne sont jamais copiées.

Les listes à puces d'un fichier Microsoft Word ne fonctionnent pas.

Si la première ligne du texte copié est formatée (un titre par exemple), ce formatage ne sera pas collé dans la messagerie CAMAP. Pour éviter ce problème vous pouvez ajouter une ligne vide en début de texte avant de le copier.

Les sauts de lignes ne sont pas pris en compte quand un message est copié depuis Google Doc.

Réutiliser un message précédemment envoyé

Il est désormais possible de réutiliser le corps d'un message précédent, et ce autant au sein d'un même groupe qu'entre groupes dont vous êtes administrateur. Pour cela, vous trouverez une barre défilante en dessous du corps du message. Cela vous permettra d'accélérer l'envoi d'un même message dans plusieurs groupes et vous empêcher les erreurs dues au copier/coller depuis une source tierce (Google Doc, Word, etc).

Seuls le sujet et le message sont réutilisés (pas les destinataires ni le nom et courriel expéditeur si jamais il a été modifié).
