Les options

Les options sont importantes puisqu'elles configurent la feuille d'inscription pour rejoindre votre groupe:
Le numéro de téléphone utile qui apparaîtra sur la feuille d'émargement si le membre arrive en retard
L'adresse postale utile qui pourra dépanner le membre avec une livraison à domicile

Le réglage des inscriptions

Fermé: inscription impossible pour un visiteur extérieur
Liste d'attente: le visiteur s'inscrit sur une liste d'attente avec les coordonnées + les options que vous avez rendues obligatoire, l'administrateur du groupe reçoit une notification et le visiteur apparaît dans l'onglet “membres” dans la colonne de droite ( liste d'attente ) en attendant votre acceptation
