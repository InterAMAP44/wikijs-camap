Propriétés du groupe CAMAP

         Texte de la page publique:   C'est le texte qui apparaît pour les visiteurs extérieurs au groupe comme ici: https://wiki.amap44.org/fr/app/page_publique

         Texte sur la page d'accueil:  C'est le texte qui apparaît sur la page d'accueil pour les membres du groupe.

         Texte listes d'émargement:  C'est le texte qui apparaît sur toutes les feuilles d'émargement

 

Les options

Les options sont importantes puisqu'elles configurent la feuille d'inscription pour rejoindre votre groupe:
Le numéro de téléphone utile qui apparaîtra sur la feuille d'émargement si le membre arrive en retard
L'adresse postale utile qui pourra dépanner le membre avec une livraison à domicile

Le réglage des inscriptions

Fermé: inscription impossible pour un visiteur extérieur. le message suivant s'affiche sur la page publique “Contactez le responsable pour vous inscrire à ce groupe et connaître les modalités d'adhésion” le bouton “Demande d'adhésion" permet au visiteur d'envoyer un mail à l'adresse du contact principal
Liste d'attente: le visiteur s'inscrit sur une liste d'attente avec les coordonnées + les options que vous avez rendues obligatoires, l'administrateur du groupe reçoit une notification et le visiteur apparaît dans l'onglet “membres” dans la colonne de droite ( liste d'attente ) en attendant votre acceptation 
Ouvert: tout visiteur extérieur peut s'inscrire en fournissant nom, prénom, mail + les options que vous avez rendues obligatoires, le visiteur devient membre sans commande s'il ne souscrit à aucun contrat, les membres sans commande apparaissent dans l'onglet “Membres ” dans la colonne de droite ( membre sans commande ) 
Complet: sur la page publique apparaît le message: “Ce groupe est complet et n'accepte plus de nouvelles inscriptions”

Contact principal

C'est un des membres avec les droits administrateurs du groupe CAMAP,  cette personne est inscrite sur le groupe Hébergement Camap / interAMAP44 , c'est la personne référente qui est en contact avec l'hébergeur interAMAP44, elle reçoit notamment les mails relatif à l'hébergement de CAMAP sur le serveur de l'interAMAP44.

Représentant légal

Le représentant légal peut être 
- soit le président de votre association classique loi 1901  
- soit le représentant légal de votre association collégial loi 1901
- soit une personne désignée par votre association de fait
C'est la personne responsable qui cautionne le bon usage des conditions générales d'utilisation définis ici: https://www.amap44.org/cgu-camap/
