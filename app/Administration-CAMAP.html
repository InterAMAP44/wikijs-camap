CAMAP: le mode AMAP de Cagette

Le mode AMAP anciennement disponible sur cagette.net est désormais hébergé sur CAMAP. 

Vous retrouverez toute la documentation utile concernant ce mode ici.

Pour comprendre le mode AMAP, il vous faut :

  • comprendre le paramétrage d'un contrat classique
  • comprendre le paramétrage d'un contrat variable
  • comprendre la gestion des absences
  • comprendre la gestion des stocks spécifiques aux AMAP

Ensuite, vous pouvez découvrir quelques cas pratiques au travers des webinaires réalisés par Alilo (développeur initial du projet) sur le sujet.
