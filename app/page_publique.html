La Page Publique de votre AMAP

La page publique est la vitrine de votre groupe CAMAP sur le web pour un visiteur extérieur à votre AMAP, 
un visiteur qui tape “camap” sur un moteur de recherche il tombe sur ce lien https://camap.amap44.org/group/map

Taper le code postal de la commune pour trouver votre page publique CAMAP:

- une carte qui géolocalise votre lieu de distribution: https://wiki.amap44.org/fr/app/admin-lieu
- l'adresse mail pour vous contacter ( contact principal : https://wiki.amap44.org/fr/app/propri%C3%A9t%C3%A9s-du-groupe
- une description de votre AMAP à définir dans les propriétés du groupe CAMAP: https://wiki.amap44.org/fr/app/propri%C3%A9t%C3%A9s-du-groupe
- un moyen d'inscription ( ouvert, liste d'attente ou complet ): https://wiki.amap44.org/fr/app/admin-gestion-produits
- une page des 5 prochaines dates de distribution issues de votre calendrier des distributions 
- une page des produits et des producteurs avec les l'affichage les prix en option : https://wiki.amap44.org/fr/app/admin-gestion-produits
- des documents relatifs à votre AMAP au format PDF: https://wiki.amap44.org/fr/app/admin-documents  
- des images média sous la carte:  https://wiki.amap44.org/fr/app/medias
