La Page Publique de votre AMAP

La page publique est la vitrine de votre groupe CAMAP sur le web pour un visiteur extérieur à votre AMAP, 
un visiteur qui tape “camap” sur un moteur de recherche il tombe sur ce lien https://camap.amap44.org/group/map

Taper le code postal de la commune pour trouver votre page publique CAMAP:

- une carte qui géolocalise votre lieu de distribution
- l'adresse mail pour vous contacter ( contact principal )
- une description de votre AMAP à définir dans les propriétés du groupe CAMAP
- un moyen d'inscription ( ouvert, liste d'attente ou complet )
- une page des 4 prochaines dates de distribution issues de votre calendrier des distributions 
- une page des produits et des producteurs avec les l'affichage les prix en option 
- des documents relatifs à votre AMAP au format PDF
- des images média sous la carte
