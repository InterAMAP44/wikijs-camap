La Page Publique de votre AMAP

La page publique est la vitrine de votre groupe CAMAP sur le web pour une personne qui n'est pas membre de votre AMAP: https://camap.amap44.org/group/map

Paramètres / Modifier les propriétés

Les dates, horaires et lieux avec géolocalisation des 4 prochaines distributions avec le nom des contrats “catalogues”

Sur cette page publique tous les producteurs de l'AMAP et leurs produits apparaît avec leurs prix en option
