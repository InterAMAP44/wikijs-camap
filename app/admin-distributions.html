Les Distributions

L'onglet “distributions” permet d'organiser les distributions d'un point de vue global dans votre groupe CAMAP. Seuls l'administrateur et les personnes ayant le droit “Gestion de tous les catalogues” peuvent y accéder.

En tant qu'administrateur de groupe, c'est à vous de planifier toutes les distributions pour toute la saison. Les producteurs et coordinateurs de catalogues/contrats pourront choisir de s'y rattacher ou pas.

Ce tableau de bord vous permet d'avoir les informations clé d'un seul coup d’œil :

Statut de la commande : Est ce que la commande est ouverte ou fermée ? ( applicable aux Contrat AMAP variables uniquement )

Producteur participants Le bouton “producteur participants” permet d'avoir la listel des producteurs présents par date de distribution. Il est précisé aussi si un de ces producteurs a défini une fermeture de commande ou ouverture de commande différente de la distribution principale.

Rôles des bénévoles : vérifier les rôles nécessaires pour aider les producteurs à la distribution 

Bénévoles inscrits : vérifier si on a suffisamment de bénévoles inscrits aux permanences. Cocher les rôles à pourvoir ou modifier la liste des inscrits.

Commandes : Nombre de commandes et total de commande

 

Comment créer des dates de distributions

Seuls les administrateurs du groupe peuvent créer les dates générales de vos distributions pour l'ensemble des producteurs (comme pour un marché de plein vent) :

Dans l'onglet “Distributions”, vous pouvez créer des cycles de distributions ou des distributions ponctuelles (une par une) à condition de ne pas avoir déjà une date de distribution ponctuelle qui interfère avec le cycle, sinon faire un cycle avant et/ou après cette date ponctuelle ( cela va créer une série de distributions ).

Une fois vos distributions créées, vous verrez le bouton “Ajouter des catalogues à la distribution”. Vous pouvez ainsi sélectionner quels producteurs participent à la distribution.

Attention si on est coordinateur de contrat/catalogue :     Vous ne pouvez pas créer de dates de distribution à proprement parler, puisque seul l'administrateur peut le faire.

Aller dans l'onglet “catalogues”, et cliquer sur “gérer” pour le catalogue concerné. Dans la page “distributions”, vous verrez toutes les dates de distribution planifiées par l'administrateur. Vous pouvez alors seulement cliquer sur “participer” pour choisir à quelles dates le producteur participera.

Comment modifier des distributions

Si vous souhaitez modifier les jours/horaires d'une seule distribution, vous pouvez cliquer Modifier.

Si vous souhaitez modifier tout un cycle de distribution, vous devez cliquer sur “Cycles de distributions”, puis “Modifier” sur le cycle concerné, afin de renseigner les nouvelles données.

Comment supprimer des distributions

Si on est administrateur du groupe :

Dans l'onglet “Distributions”, cliquer sur “producteurs participants” sur la distribution concernée. Vous pouvez cliquer sur “ne participe plus” pour tous les catalogues que vous souhaitez retirer de cette distribution.

Une fois que vous arrivez à zéro catalogues participants, vous pouvez éventuellement supprimer la distribution avec le bouton “Suppr” 

Vous pouvez également supprimer tout un cycle de distribution en cliquant sur “Cycle de distribution” et “Supprimer”.

Si on est coordinateur de contrat/catalogue :

Aller dans l'onglet “catalogues”, et cliquer sur “gérer” pour le catalogue concerné. Dans la page “distributions”, vous verrez toutes les dates de distribution planifiées par l'administrateur. Vous pouvez cliquer sur “ne plus participer” pour annuler la participation de ce producteur à cette distribution. Notez que Camap peut refuser d'annuler la participation si il y a des commandes enregistrées pour ce producteur à cette date.
